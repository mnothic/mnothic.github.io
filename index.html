mdbuilder

Este proyecto fue creado como agente escrito en ISO C sobre FreeBSD, para mantener una plataforma de correo distribuida cuya base de datos de usuarios virtuales y MTA son independientes entre si tanto física y lógicamente hablando. Ademas permite integrar un esquema de base de datos diseñado para funciones de la organización e integrarlo con otras aplicaciones.
